User Guide

Expenses / Treasury & Operations Ledger

Maly - OBS (Online Business System) · Version 1.0

Module Purpose

The Expenses module tracks income and expenses, maintains the treasury (cash) balance, and records financial operations in a date-filtered ledger. You can add manual entries (income or expense), view totals by category, close the day with an actual cash count to reconcile differences, and consolidate all transactions into summary entries. The module also provides analytics to see how expenses break down by category. Treasury balance is strictly enforced: you cannot record an expense that would make the cash balance negative.

Step-by-Step Workflow

1. Add a New Entry (Income or Expense)

  • Click New Entry.
  • Choose Income or Expense (Expense is default).
  • Select a Category from the dropdown, or click the + icon to add a new category.
  • Enter the Amount (e.g., 150.50).
  • Optionally enter Notes.
  • Click the Date button to pick a date (defaults to today for new entries).
  • Click Save.
  • The dialog closes. If saved successfully, a green snackbar shows; the ledger and KPIs refresh. If the amount is invalid or required fields are missing, the dialog stays open (fix and try again).

2. Edit an Entry

  • In the ledger, find the row and click the Edit icon (blue pencil).
  • Change Category, Amount, Type, or Notes. Date cannot be changed when editing.
  • Click Save.
  • Success: green snackbar; ledger refreshes.
  • Note: System-generated entries (e.g., daily closing adjustments, consolidated entries) show a "System" badge instead of Edit/Delete and cannot be edited.

3. Delete an Entry

  • In the ledger, click the Delete icon (red trash).
  • Confirm in the dialog.
  • Success: green snackbar; ledger refreshes.
  • Note: System-generated entries cannot be deleted.

4. Add a New Category

  • Open New Entry or Edit.
  • Click the + (plus) icon next to the Category dropdown.
  • Enter the category Name and click Add.
  • The new category is created and selected. Continue filling the entry form and click Save.

5. Change Date Range

  • Click the Date Range field (shows start → end).
  • In the dialog, choose a preset (Today, Yesterday, This Week, Last Week, This Month, Last Month, This Year, Last Year, All Time) or pick custom Start and End dates.
  • Apply the selection. The ledger and KPIs update to the new range.
  • The range is remembered for the session.

6. Search in the Ledger

  • Type in the Search field.
  • The ledger filters by category, notes, or amount in real time.
  • KPIs still reflect the full filtered result set.

7. Close the Day (Reconcile Cash)

  • Count your actual cash on hand.
  • Click Close Day.
  • See the System Balance (calculated from all income and expenses).
  • Enter the Amount you actually have (actual cash).
  • Click Confirm.
  • If the difference is more than 0.01, the system adds a protected Adjustment entry (Income or Expense) to match the actual balance. A green snackbar confirms with the difference shown.
  • Note: Actual cash cannot be negative. If you enter a negative value, an error message appears.

8. Consolidate Transactions

  • Click Consolidate in the header.
  • Read the warning: all transactions (manual and system) will be grouped by category into single entries; consolidated entries cannot be edited; this cannot be undone.
  • Click Confirm to proceed or Cancel to close.
  • Success: green snackbar; ledger refreshes with fewer, grouped entries.
  • If there are no transactions: "No transactions to consolidate" is shown.
  • Note: Consolidation affects all transactions in the system, not only the current date range.

9. View Analytics

  • Open the Analytics & Reports tab.
  • See the Expenses Breakdown chart (categories with amounts and progress bars).
  • Toggle Include Purchases & System to include or exclude purchase and system-generated transactions in the chart.
  • The chart shows only expense categories (no income).

10. Paginate the Ledger

  • Use Previous and Next to move between pages (15 rows per page).
  • The label shows "Page X of Y".

Tips

  • Treasury shows total cash; it turns red if the balance is negative.
  • System entries (adjustments, consolidated entries) are protected and cannot be edited or deleted.
  • Date can only be set when creating a new entry; editing keeps the original date.
  • Include Purchases & System in Analytics lets you see purchase-related categories in the breakdown; off by default for a cleaner view.
  • Consolidate reduces many transactions to one per (category, type). Use when you want a simplified ledger for reporting.
  • Close Day is for daily reconciliation; enter the actual cash you have so the system can adjust for differences.
