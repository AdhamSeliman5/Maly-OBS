Manage Courier Orders & Collections

The Shipping module manages orders handed over to a courier. Select a company, view active (in-transit) orders, update delivery status (Delivered, Partial, Returned, Damaged), adjust shipping fees and extra costs, then process collection to finalize. Track loans (advances) with couriers, generate collection reports, and export PDF analytics. Net settlement uses total minus shipping fees.

  • KPI cards: Active Orders, Pending Amount, Loan, Net Due, Delivery Rate
  • Update Status, Ship Cost, Extra (Shipped on Sender / compensation), Delivery Date
  • Process Collection, Ledger (Add Loan / Repay), Export PDF report

Capabilities

What You Can Do

Key features at your fingertips.

Courier Management

Add or delete shipping companies. Select a courier to load active orders, ledger, and analytics. Search orders by ID, customer, phone, governorate.

Order Status Updates

Edit Ship Cost (Net recalculates). Set Status: Delivered, Partial, Returned, Damaged. For Partial: enter delivered quantities and unit prices in the dialog.

Collection & Ledger

Check orders, click Process Collection. Dialog shows Net Amount, Cash Received, Unified Delivery Date. Generates Excel report. Add Loan (advance) or Repay from Ledger tab.

PDF Analytics

Export PDF icon in header. Generates shipping report for selected company. Saves to Desktop\Reports\Shipping\ and opens automatically.

Workflow

How It Works

Get started in minutes.

1

Select a Shipping Company

Use the dropdown to choose a courier. The system loads active orders, ledger, and analytics. Add new companies with the green + icon.

2

Update Order Status and Amounts

In Active Orders tab: edit Ship Cost, set Status (Delivered, Partial, Returned, Damaged). For Returned/Damaged, fill Extra (Shipped on Sender or compensation). Pick delivery date.

3

Process Collection

Check the orders to settle. Click Process Collection. Set Cash Received (default = Net), Unified Delivery Date, Stale Days. Click Process. Excel report generates and opens.

4

Add or Repay Loans

Ledger tab: Add Loan (positive = courier gives advance, negative = you give advance). Click Repay on unpaid loans to settle. Treasury must be sufficient for payouts.
